What This Site Tracks

This website is an educational project to demonstrate what information a typical website can log about its visitors using only browser-based technologies (JavaScript).

  • Initial visit details (referrer, screen size).
  • All mouse movements and clicks.
  • Text typed into non-sensitive fields.
  • Pages you visit within this site.
